# 🌍 Country Info App

A live web app that fetches real-time country data
using a public REST API and displays it instantly.

## 🚀 Live Demo
👉 https://dhurianii.github.io/js-api-project

## What It Does
- User types any country name
- App fetches live data from REST Countries API
- Displays flag, capital, population, currency, language
- Handles errors and loading states gracefully

# 🏋️ Gym Landing Page

A professional React-based gym website with lead 
capture form, pricing plans, and form validation.

## 🚀 Live Demo
👉 https://gym-landing-ashen.vercel.app

## What It Does
- Professional gym landing page — dark gold theme
- Fixed navbar with smooth scroll
- Hero section with stats counter
- 3 pricing plans — Basic, Pro, Elite
- Lead capture form with modal popup
- Form validation — name, phone, goal
- Success screen after form submission

## ⚛️ React Concepts Used
- Functional components
- Props — data passing between components
- useState — form data + modal visibility
- Conditional rendering — &&, ternary, early return
- Event handlers — onClick, onChange
- Array.map() — rendering pricing cards

## 📋 Components Built
| Component | What it does |
|-----------|-------------|
| Navbar | Fixed top navigation with gold logo |
| Hero | Main banner with stats and CTA button |
| Plans | 3 pricing cards with popular badge |
| LeadForm | Modal form with full validation |

## ✅ Form Validation Rules
- Name — minimum 2 characters required
- Phone — exactly 10 digits, Indian format (6-9 start)
- Phone — only numbers allowed, maxLength=10
- Goal — must be selected before submit
